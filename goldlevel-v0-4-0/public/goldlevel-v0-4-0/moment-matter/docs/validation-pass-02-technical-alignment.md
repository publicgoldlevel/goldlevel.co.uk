# Validation Pass 02 — Technical Alignment

- Static navigation is present across all pages.
- Product data is centralized in `assets/data/products.js`.
- Product grid renders from data.
- Product detail pages resolve by query string: `product.html?id=...`.
- Basket uses local browser storage.
- Checkout creates a copyable order request and does not pretend to collect live payment.
- Shipping estimates distinguish digital from physical and UK from worldwide.
- Original project artwork is embedded in `assets/img/originals/`.
- Optimized artwork is embedded in `assets/img/artwork/`.
- Previous archives are preserved in `docs/legacy_archives/`.
- Source material is preserved in `docs/source_material/`.
