# Lived Alchemy — Stage 09: Projection

**Core movement:** revealed  
**On-slide text:** Projection reveals the lens.  
**Visual anchor:** Warped mirror pool, horizon, shadow, solitary figure.

## Image Creator Prompt

Create one vertical 4:5 cinematic fine-art image for a coherent visual sequence titled **Lived Alchemy**. Use the same restrained elemental visual language across the sequence: dark earth, black water, stone, aged manuscript, fine gold trace, ash, clay, glass, weather, horizon. Make the image symbolic, material, and museum-grade. Do not include readable text, logos, or watermarks.

Symbolic landscape: solitary figure facing a vast terrain reflected in a slightly warped black mirror pool, reflection subtly different from the real horizon, long shadow bending across stone ground, restrained dusk blue black gold palette, contemplative, mysterious, no text.

## Negative Prompt

Avoid horror mirror imagery, narcissistic portrait focus, obvious split-screen, psychological cliché, surreal face distortion, accusatory mood.

## Caption Fragment

Projection reveals the lens. The world becomes information without becoming accusation.

## Regeneration Controls

If too literal: make it more symbolic, quieter, less illustrative, less crowded.  
If too mystical: make it more grounded, material, editorial, and restrained.  
If visually disconnected from the sequence: increase continuity through black water, stone, manuscript texture, and a fine gold trace.
