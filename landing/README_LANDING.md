## GitHub Pages

Recommended repository structure:

```text
repo-root/
  docs/
    index.html
    assets/
```

In GitHub:

1. Open repository Settings.
2. Open Pages.
3. Set Source to `Deploy from a branch`.
4. Select branch `main` and folder `/docs`.
5. Save.
