## Publish steps
1. Request email is set to `info@goldlevel.co.uk`.
2. Open `index.html` in a browser and review all copy.
3. Upload the folder to any static hosting route.
4. Test every CTA.
5. Send yourself one test request from the intake form.
6. Record the URL and first test receipt.
