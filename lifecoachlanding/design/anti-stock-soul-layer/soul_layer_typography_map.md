# Soul Layer Typography Map v0.1.0

## Display voice

Used for:

- hero line;
- emotional section headings;
- bridge statements;
- human transformation phrases.

Recommended stack:

```css
"Cormorant Garamond", "EB Garamond", Georgia, "Times New Roman", serif
```

## Structure voice

Used for:

- cards;
- buttons;
- form fields;
- body copy;
- navigation.

Recommended stack:

```css
Inter, Manrope, "Segoe UI", system-ui, sans-serif
```

## Micro-label voice

Used for:

- eyebrow labels;
- protocol labels;
- scene kickers;
- status markers.

Rules:

- uppercase;
- letter spacing 0.14em–0.18em;
- small but legible;
- gold/sage rather than black where possible.
