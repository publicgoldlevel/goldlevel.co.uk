## 1. Core diagnosis

The site now functions, but the visual language can still read as stock because images, cards, text, and sections remain too separable.

The next design correction is not another isolated component patch. It is an authored layer that governs:

- typography cadence;
- gradient atmosphere;
- image treatment;
- micro-elements;
- section continuity;
- low-motion interaction.

## 2. Operating thesis

```text
Not a website with assets inserted into it.
A composed visual field where image, text, gradient, and proof belong to one authored surface.
```

## 3. Anti-stock rules

### Rule A — Image is not illustration only

Every WebP should be treated as one of:

- visual field;
- ambient field;
- symbolic texture;
- scene anchor.

### Rule B — Typography must carry emotional cadence

Use three voices:

- display serif for emotional authority;
- structural sans for scannable sections;
- micro-label caps for protocol/order.

### Rule C — Small authored elements matter

Use subtle halos, rings, dividers, flow lines, and proof marks to make the design feel intentional rather than template-driven.

### Rule D — No hidden claim inflation

A more premium interface must not imply market traction, software maturity, client proof, or conversion validation without receipts.
