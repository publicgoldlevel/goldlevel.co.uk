# Product Artifact Image Folder

This folder is wired into the website. To replace any generated asset, keep the exact filename below and overwrite the file in this folder.

Required filenames:

- `custom_moment_card_digital.png` — Custom Moment Card — Digital
- `custom_moment_card_physical.png` — Custom Moment Card — Printed & Posted
- `custom_moment_map_digital.png` — Custom Moment Map — Digital
- `custom_moment_map_digital_example.png` — homepage / example Moment Map preview
- `Custom_Moment_Map_printed.png` — Custom Moment Map — Printed
- `clarity_map_digital.png` — Clarity Map — Digital
- `CLARITY_MAP_Print_edition.png` — Clarity Map — Print Edition / premium wide hero

The site uses `assets/data/products.js` and `assets/js/site.js` to resolve these images. The premium print edition is intentionally treated as a wide landscape hero image rather than a small card preview.
